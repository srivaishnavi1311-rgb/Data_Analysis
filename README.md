# Data_Analysis
Developed an interactive Sales Analysis Dashboard in Microsoft Excel using Pivot Tables, Pivot Charts, and Slicers to analyze sales trends, customer behavior, regional performance, and sales channel contribution for data-driven business insights.
# Project Overview

Developed an interactive Sales Analysis Dashboard in Microsoft Excel to analyze sales performance, customer behavior, and business trends. The dashboard leverages Pivot Tables, Pivot Charts, Slicers, and Excel formulas to transform raw sales data into actionable business insights.

# Business Objective

Analyze sales data to identify trends, customer preferences, regional performance, and channel effectiveness, enabling data-driven business decisions.

# Key Insights
Women contributed the highest share of sales.
Adult customers generated the most orders.
Most orders were successfully delivered.
Maharashtra recorded the highest sales.
Amazon was the leading sales channel.
Interactive filters enable quick and flexible analysis.
